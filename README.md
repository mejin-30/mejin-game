## 游戏介绍

《古籍密码》是一款融合中国古代神话与现代刑侦元素的网页解密游戏。玩家将扮演一名文物犯罪调查科警员，调查故宫博物院珍贵古籍《永乐大典》抄本失窃案。通过分析线索、破解密码和调查嫌疑人，揭开案件背后隐藏的超自然秘密。

游戏采用多网页嵌套结构，结合多种密码破解机制和多结局系统，提供约3-5小时的沉浸式推理体验。

## 玩法说明

### 基本操作

1. **开始游戏**：打开 `index.html` 文件开始游戏
2. **查看线索**：点击左侧"案件档案"中的线索展开详情
3. **搜索功能**：使用顶部搜索栏输入关键词查找相关线索
4. **破解密码**：遇到密码谜题时，根据提示进行破解
5. **记录笔记**：点击顶部"笔记"按钮记录重要线索和密码
6. **返回功能**：使用顶部"返回"按钮回到上一级页面

### 密码破解机制

游戏包含多种密码破解环节：

- **摩斯密码**：通过点(.)和划(-)的组合来表示字母
- **凯撒密码**：通过将字母表中的每个字母按照固定数量进行偏移来实现加密
- **洛书密码**：基于中国古代洛书的数字排列规律

## 故事背景

故宫博物院发生一起离奇案件，一卷珍贵的明代古籍《永乐大典》抄本在深夜被盗。古籍保管室门窗完好，无明显闯入痕迹，监控录像在案发时段出现异常干扰。唯一的线索是现场留下的一张神秘纸条，上面写着奇怪的符号和一个红色的"血"字。

随着调查深入，玩家发现这起案件与中国古代神话中的"洛书"有关，失窃的古籍中记载着一种可以唤醒超自然力量的仪式。玩家需要通过分析线索、破解密码，揭开案件背后的真相。

## 功能特点

- **多网页嵌套结构**：点击线索展开新页面，提供沉浸式体验
- **关键词搜索系统**：支持模糊搜索，帮助玩家查找相关线索
- **多结局系统**：根据玩家选择导向不同结局
- **第四面墙打破效果**：增强游戏沉浸感
- **中国古代神话元素**：融合洛书、女娲补天等神话元素
- **东野圭吾式反转剧情**：情节跌宕起伏，充满反转
- **恐怖氛围营造**：适当使用灰色与红色，营造压抑恐怖氛围

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript

## 游戏攻略

1. **仔细阅读文本**：游戏中的线索和提示往往隐藏在文本中
2. **注意细节**：图片和文本中的细节可能包含重要线索
3. **记录重要信息**：使用笔记功能记录重要线索和密码
4. **联想思考**：将不同线索联系起来，进行推理
5. **尝试不同组合**：遇到密码谜题时，尝试不同的组合和可能性
